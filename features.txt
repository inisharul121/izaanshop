# Izaan Shop - Implemented Features

## 1. Authentication & User Management
- **User Registration & Login**: Standard user authentication with JWT.
- **Admin Portal**: Separate login/registration for administrators.
- **Admin Approval System**: New admin accounts require approval from an existing admin before access is granted.
- **Stale Session Handling**: Global 401 interceptor to auto-logout and redirect users when sessions expire or roles change.
- **Role-Based Access Control (RBAC)**: Secure routes for 'user' and 'admin' roles.

## 2. Product Management
- **Dynamic Catalog**: Full CRUD operations for products, categories, and coupons.
- **Variable Products**: Support for product attributes (e.g., Color, Size) and variants with independent SKUs, prices, and stock.
- **Image Management**: Integrated Media Library for uploading and reuse of images.
- **Product Search & Filtering**: (Implicitly supported by the catalog structure).

## 3. Shopping Experience
- **Cart System**: local storage persistent cart with real-time updates.
- **Checkout Process**: Multi-step checkout with guest checkout support.
- **Guest Checkout**: Ability for users to place orders without creating an account.
- **Form Validation**: Zod-powered client-side validation for checkout and auth.

## 4. Order & Payment Management
- **Order Tracking**: Detailed order history for users.
- **Admin Order Dashboard**: Comprehensive view for admins to manage all orders.
- **Order Details View**: Detailed modal for admins to see specific items, customer shipping info, and price breakdowns.
- **Delivery Status**: "Mark Delivered" functionality in the admin panel.
- **Payment History**: Payments tab tracking transactions and customer names (including guest names).

## 5. UI/UX & Design
- **Premium Design**: Modern, responsive UI with sleek dark modes and vibrant color palettes.
- **Interactive Elements**: Micro-animations using Framer Motion.
- **Mobile Responsive**: Fully optimized for mobile and tablet devices.
- **Iconography**: Clean, consistent icons from Lucide React.

## 6. Technical Stack
- **Frontend**: React, Vite, Zustand (State Management), Tailwind CSS.
- **Backend**: Node.js, Express, MySQL.
- **ORM**: Prisma for type-safe database interactions.
- **Version Control**: Git integration.
